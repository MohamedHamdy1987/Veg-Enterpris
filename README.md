# نظام إدارة الخضار والفواكه - Enterprise

نظام SaaS متكامل لإدارة محلات الخضار والفواكه. يدعم تعدد الفروع، صلاحيات المستخدمين، إدارة المخزون، المبيعات النقدية والآجلة، التحصيلات، المصروفات، والتقارير.

## 🚀 المميزات
- نقطة بيع سريعة.
- تتبع المخزون مع تنبيهات.
- حسابات العملاء والموردين.
- متعدد الفروع مع عزل بيانات كامل.
- نظام اشتراكات (شهري/سنوي).
- تقارير مالية ورسوم بيانية.
- واجهة عربية بالكامل.

## ⚙️ المتطلبات
- حساب على [Supabase](https://supabase.com).
- متصفح حديث (يدعم ES6).
